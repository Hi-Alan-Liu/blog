前言

Visual Studio Code (簡稱 VSCode) 是一款由微軟開發且跨平台的原始碼編輯器。VSCode 體積小、可跨平台、還可以根據需求加入自己喜好的插件。
而好的插件能大大提升自己 Coding 的速度，美美樣式及酷酷的圖標還能療癒身心(?)
