Windows 防火牆新增輸入規則

Windows 防火牆新增輸入規則

前言

因在 Windows 上佈署伺服器，但一般除了預設Port 80，443 其餘是不開放的。

所以將自行設定防火牆輸入規則

開始

設定輸入規則

進入 Windows Defender 防火牆

選擇進階設定

選擇輸入規則並選擇新增規則

選擇連接埠

  • 埠 (ㄅㄨˋ (又音) ㄈㄨˋ)

選擇規則 TCP 或 UDP，我這邊是供 Web 使用所以選擇 TCP

填入特定的連接住埠(Port)

選擇連線條件，我這邊選擇所有連線

設定名稱及其描述

點選完成即可完成設置

補充、設定白名單

如果想要設定連線白名單

可以在設定動作時，選擇僅允許安全連線

之後授權的使用者加入在此處即可完成
